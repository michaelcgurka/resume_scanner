# Tech Role Resume ATS Software
### This web app mimics the software that tech companies use in their application pipelines to auto-filter resumes that align well with the skills and experiences they are looking for.
##### Big picture: upload PDF of resume and job description as text -> parse information and upload -> score the resume based on JD-matching/NLP/predefined metrics (GPA) -> output resume score on web page with other features and suggestions
<u> Tech Stack </u>
* JavaScript
* React.js
* Python
* PostgreSQL
