# Tech Role Resume ATS Software
### This project aims to mimic the software that tech companies use in their application pipelines to auto-filter resumes that align well with the skills and experiences they are looking for.
##### This is still in progress (MVP) but will upload, process, and score resumes with output on the same webpage with insights for personal resumes!




<u> Main Tech Stack </u>
* JavaScript
* React.js
* Python
* PostgreSQL
